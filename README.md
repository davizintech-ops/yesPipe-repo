# 📱 yesPipe (baseado no notpipe)

Uma versão modificada, otimizada e com tema escuro do **NotPipe**, projetada especificamente para rodar com o máximo de desempenho em dispositivos antigos (como o icônico Galaxy Ace Duos) ou para quem busca economia extrema de memória.

---

## 🚀 Principais Otimizações e Recursos
* **Fundo Escuro Nativo:** Interface modificada em AXML para reduzir o consumo de bateria e melhorar o visual.
* **Player Assíncrono Padrão:** Opção de URI Assíncrona pré-ativada para evitar travamentos e congelamentos ao carregar mídias.
* **Conversão Automática de Codec:** Força a compatibilidade de vídeo por padrão, corrigindo os erros de reprodução comuns em Androids antigos.
* **Ultra Leve:** Limpeza de arquivos inúteis e recursos otimizados direto na compilação do APK.

## 📥 Como Baixar (Downloads Oficiais)

Sempre baixe o aplicativo direto da aba de lançamentos oficiais do GitHub para garantir que você está com a versão mais recente e segura.

👉 Baixe o APK clicando **[aqui](https://github.com/davizintech-ops/yespipe-repo/releases)**.

---

## 🛠️ Como Instalar
1. Vá na aba **Releases** no link acima.
2. Baixe o arquivo `.apk` da versão mais recente.
3. Se o seu Android bloquear, ative a opção *"Fontes Desconhecidas"* nas configurações de segurança do aparelho.
4. Instale e aproveite!
